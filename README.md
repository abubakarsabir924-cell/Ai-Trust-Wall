# 🛡️ Ai-Trust-Wall
*An AI-assisted trust verification system for secure environments.*

## 🚀 Overview
Ai-Trust-Wall is a security solution designed to monitor and verify user integrity in remote environments like online exams and interviews. It uses AI to detect unauthorized activities in real-time.

## ✨ Key Features
* *AI Monitoring:* Real-time behavior analysis.
* *Integrity Checks:* Prevents bypass techniques.
* *Instant Alerts:* Automated flagging of suspicious actions.

## 🛠️ Tech Stack
* *Language:* Python
* *Backend:* Flask / FastAPI
* *Concepts:* Computer Vision, Security Workflows

## ⚙️ How to Run
1. Clone the repo: git clone https://github.com/abubakarsabir924-cell/Ai-Trust-Wall.git
2. Install requirements: pip install -r requirements.txt
3. Start the app: python app.py
